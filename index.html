Quiz Game

SOME INFORMATION AND RULES ABOUT THE GAME ARE GIVEN BELOW:

  • ALL questions are Multiple Choice.
  • There are 7 questions in total. [Score: 7 x 2 = 14]
  • Right answer: 2 points / Wrong answer : -1 from the points.
  • Passing mark 11 or above 11/ less than 11 Fail.
  • Each question carries 2 points.
  • Answering all the questions correctly to win the game.
